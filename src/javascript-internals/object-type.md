객체 타입 (Object type)
===
컴퓨터 과학에서의 객체는 "식별자"로 참조할 수 있는 메모리 상의 값을 말함.   
JS에서 객체는 속성의 컬렉션으로 볼 수 있고 변경 가능한 값임.   

아래는 객체 타입(배열)이 어떻게 참조되는 지에 대한 예제임.
```javascript
let arr1 = [1, 2, 3];
let arr2 = arr1;
arr1[0] = 99;

console.log(arr1); // [99, 2, 3]
console.log(arr2); // [99, 2, 3]
```
```
Stack:
┌───────┐
│ arr1  │ ─────────┐
└───────┘          │
│ arr2  │ ─────────┘
└───────┘

Heap:
┌────────────┐
│ [99, 2, 3] │  <- arr1과 arr2가 같은 객체를 참조
└────────────┘

변수 arr1, arr2는 스택 영역에 주소(참조)를 저장함.
힙 영역에 저장된 데이터를 변경하여 두 변수 모두 변경 사항을 확인 가능함.
```
